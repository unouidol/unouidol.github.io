Data & Account Control

Information about in-app controls and app-related data management

TwitchMiniChat provides tools and flows to help users manage app-related data, safety preferences, and deletion requests.

Available controls

  • Block users in the app
  • Report messages for moderation review
  • Clear app data stored on the device
  • Request deletion of app-related server-side data

Local data

Some settings are stored locally on the device, such as preferences, blocked users, and app state needed for app functionality.

Server-side data

Some app-related data may be stored server-side to support features such as chat history, push notifications, moderation review, and data-control requests.

Deletion requests

Users can request deletion of app-related data through the official data deletion page. See Data Deletion.

Retention note

Some limited records may be retained for a short time, where reasonably necessary for security, abuse prevention, fraud prevention, audit, moderation review, or legal compliance.
