Report & Block

In-app safety tools for TwitchMiniChat

TwitchMiniChat provides in-app safety tools for public Twitch chat content.

How to report a message

In chat, long-press a message and tap Report message.

How to block a user in the app

In chat, long-press a message and tap Block user in app. Messages from blocked users are hidden inside TwitchMiniChat.

What a report may include

  • Message text
  • Username
  • Channel
  • Message ID, if available
  • Timestamp

Why these tools exist

These tools help reduce abuse, harassment, and objectionable content, and support moderation review.

Important note

Blocking a user in TwitchMiniChat hides that user’s messages inside the app. It does not replace Twitch’s own moderation or blocking tools.
